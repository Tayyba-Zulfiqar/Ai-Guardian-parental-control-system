# 🌸 AI Guardian — Parent Dashboard 💻✨

Welcome to the **Parent Dashboard** of the *AI Guardian* system 🛡️💖
This website helps parents stay aware, informed, and in control of their child’s digital world 🌍📱

---

## 🧁 About This Project

The Parent Dashboard is a web app where parents can:

* 👀 View their child’s screen time
* 📊 Track app usage
* 🚨 Get alerts for harmful activity
* ⚙️ Manage controls and restrictions

All in a clean, calm, and easy-to-use interface 🌈✨

---

## 🛠️ Tech Stack

* 💛 JavaScript
